# ChatAja Jobs (Front-End)

Sebuah portal untuk melakukan lamaran dan publikasi lowongan pekerjaan. Disediakan untuk job seeker dan employer.

## Getting Started
Instruksi di bawah adalah untuk instalasi, menjalankan di local server

### Prerequisites
``` bash
node : v11.x.x
yarn : 1.x.x
```

### Installing
``` bash
# install dependencies
$ yarn install
```

### Running Development Server
``` bash
# serve with hot reload at localhost:3000
$ yarn dev
```

### Build files for deployment
```bash
# build for production and launch server
$ yarn build
$ yarn start
```

## Built With
* Nuxt.js 2.14.6
* Vuetify 1.11.2
* Moment 1.16.1
* Nuxt Auth 5.0.0
* Dropzone Nuxt 0.1.3
